隐私权政策

本隐私权政策适用于 Chrome 浏览器扩展程序：虾皮Shopee价格翻译器（印尼站）。

1. 数据收集

本扩展程序会处理用户访问页面的部分网站内容（主要是价格文本信息），以及相关的用户活动（如浏览特定元素时触发转换）。

2. 数据使用目的

所有数据仅用于在用户本地浏览器中识别并转换印尼盾价格为人民币，不用于其他用途。

3. 数据存储

插件不会将任何数据上传至远程服务器。所有设置（如自定义汇率、API地址等）均保存在用户的本地浏览器存储中。

4. 数据共享

我们不会收集任何个人身份信息，也不会将任何数据出售、共享或传输给第三方。

5. 第三方服务

插件可能访问用户配置的汇率 API 以获取汇率数据，该行为取决于用户设置。

6. 用户控制

用户可在浏览器中禁用或卸载插件，以完全停止插件的功能和数据处理行为。

7. 联系方式

如有任何问题或建议，欢迎联系开发者：gangbigong@foxmail.com
